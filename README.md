# 🌊 Ocean - Message In A Bottle

An anonymous social experience where users can throw messages into a digital ocean and discover thoughts from strangers around the world.

## Features

* Anonymous message sharing
* No accounts or login required
* Random bottle discovery
* Ocean-themed UI
* Prevents immediate message repeats
* MongoDB storage
* React + Express + MongoDB

## Tech Stack

### Frontend

* React
* Tailwind CSS
* React Router
* Framer Motion

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## How It Works

1. Write a message.
2. Throw it into the ocean.
3. Sail through random bottles from others.
4. Discover anonymous thoughts drifting ashore.
